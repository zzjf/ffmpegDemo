*x86_64*架构的*Xcode*工程,可调试*FFmpeg*

可编译生成*FFmpeg*静态库的脚本

来自[FFmpeg-iOS-build-script](https://github.com/kewlbear/FFmpeg-iOS-build-script)

## Learn-FFmpeg

引入了*FFmpeg*静态库的*iOS*项目

## Learn-FFmpeg.xcworkspace

包含*Learn-FFmpeg*和*Debug-FFmpeg*工程
